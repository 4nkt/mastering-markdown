# Mastering Markdown

Markdown is a markup language used to write both HTML and formatted text. It is easy to read just by itself as well as it converts to HTML.

## Paragraph And Text Decoration

This is a paragraph.

This is **bold**.

This is _italic_.

This is a ~~strikethrough~~.

Heading 1
